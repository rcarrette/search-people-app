**Simple project for personal technical training purposes.**

* Node
* TypeScript
* GraphQL
* MongoDB
* Mocha

**MongoDB setup**

Assuming MongoDB is installed on your machine, just create a new database named ``` search-people-db ``` and ```api\mongodb\data``` folder.

**Dev environment setup**

```
npm install
npm run dev
```

**Run tests**

```
npm run tests
```
